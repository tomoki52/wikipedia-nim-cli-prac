# Wikisearch

a practice of Nim

## Usage: 
  wikisearch \<word> [--sort=[sort]] [-r \| --reverse] [--limit=\<limit>]

## Options:
  \<word>          Search word  
  --sort=\<sort>   Sort by pageid/wordcount [default: pageid]  
  -r --reverse    Sort by asc  
  --limit=\<limit> Limit to display [default: 10]
